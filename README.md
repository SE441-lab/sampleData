
# Textbooks Dataset (MongoDB)

This project contains a sample **MongoDB dataset of software engineering textbooks** used for demo, testing, and end-to-end pipeline exercises.

## Dataset Overview

* **Database:** `superData`
* **Collection:** `textbooks`
* **Format:** JSON array
* **Records:** 9 textbooks

Each document represents a textbook commonly used in software engineering education.

## Document Schema

```json
{
  "id": Number,
  "name": String,
  "author": String,
  "edition": String,
  "image": String,
  "description": String
}
```

> `_id` is automatically generated by MongoDB.

## Importing the Data

```bash
mongoimport --db superData --collection planets --jsonArray --file textbook.json
mongoimport --db superData --collection os --jsonArray --file os.json
```

## Example Query

```js
db.textbooks.findOne({ id: 1 })
```

## Intended Use

* Backend API demos (Node.js + Express + MongoDB)
* CI/CD and Jenkins pipeline testing
